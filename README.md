# The City Choir Manager

A Spring Boot application for managing choir member registrations, authentication, and profile management.

## Features

- User registration with email verification
- Automatic registration number generation based on choir part
- Authentication using JWT
- Profile management (view/update)
- Email notifications using Gmail SMTP
- Validation and custom exceptions for clean error handling
- Member Attendance
- Request Permission
- Admin Portal
- Ticket/Events Management

## Technologies Used

- Java 17
- Spring Boot 3.x
- Spring Security
- Spring Data JPA
- MySQL 8
- JWT (JSON Web Tokens)
- Jakarta Mail (for sending emails)
- Lombok
- Maven
